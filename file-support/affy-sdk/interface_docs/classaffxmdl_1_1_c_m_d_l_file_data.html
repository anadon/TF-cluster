Detailed Description

PLIER model file stores the model data as the structure defined above. (1) It is probe array type specific. Only the model file with the same probe array type can be used for PLIER analysis (2) It stores the model data for each probe (or cells). (3) It starts storing model data for all PM probes and follows by all MM probes. (4) Sum(a(j)) = # of probes
