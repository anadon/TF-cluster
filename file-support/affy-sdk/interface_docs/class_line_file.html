Detailed Description

Platform safe class for reading lines.

LineFile For Test Code ONLY. Chances are you should be using file/TsvFile/TsvFile and not this object. This file is intended for test code in util/ being used by file/TsvFile/TsvFile and other code not using TsvFile.
