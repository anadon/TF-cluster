Create a "multi-data" CHP file with just the header information. The remainder of the file will be created at a later time using the buffer writer technique. The CHP file will contain only "genotyping" results.

Parameters:
execIdThe execution identifier. This identifier is used to identify the batch run that created the CHP files.
celFileThe full path to the parent CEL file. The header of the CEL file is copied to the CHP file.
outFileThe name of the output CHP file.
extraColNamesThe names of the extra data columns. Should not include probe set name, call and confidence columns.
extraColTypesThe types (float, int, ubyte) of the extra columns.
numEntriesThe number of rows (entries) of results to store in the CHP file.
maxProbeSetNameLengthThe maximum length of the probe set names.
algNameThe name of the algorithm used to create the results.
algVersionThe algorithm version.
chipTypethe chip type, also known as the probe array type.
programNameThe name of the program used to create the CHP file.
programVersionThe version of the program.
