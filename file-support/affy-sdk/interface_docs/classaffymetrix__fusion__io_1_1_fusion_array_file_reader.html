Detailed Description

This class provides the fusion parsing interfaces for reading array files.

The array files can be one of the following: MAS5 EXP files MAGE-ML XML files generated by the DTT version 1.1 software MAGE-ML XML files generated by the GDAC Exporter SDK

In all cases the parser will populate an array data object.
