Detailed Description

TsvFileField holds the data read from a column. The data is read into a std::string buffer. Rather than resizing the buffer all the time, we try and fill it "in place". Most of the time the new value should be the same size as the old value. We store the clvl and cidx info in the Column as well. Saves having to search for it in the TsvFile object.

"cidx" is counted from *after* the leading tabs.
