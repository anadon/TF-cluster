InstanceOf

Check at run time if an object is an instance of a class. This differs from comparing the result of type_id, as it takes inheritance into account. Note that base class must be polymorphic, that is it must have a virtual function or inherit from another class.

Parameters:
objPtrA pointer to the object to check. A pointer must be used, not an object. If necessary, generate an address.
Theclass name to test against.
Returns:
true if the object is an instance of the class.
