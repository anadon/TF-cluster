int affxcdf::CCDFProbeSetInformation::GetNumGroups ( ) const [inline]

The number of groups in the set.

Returns:
The number of groups in the set. In genotyping arrays it is convenient to group organize probes that interrogate a polymorphic SNP into subsets that interrogate a particular allele and have the same direction (i.e., sense or antisense). Each grouping is a "probe group." Affymetrix also uses the term "block" for a "probe group."
