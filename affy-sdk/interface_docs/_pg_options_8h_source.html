00032   @section PgOptionsIntro Introduction
00033 
00034   PgOptions is the APT library for working with command line
00035   options.  With this library, a program can declare the
00036   list of options it supports along with their kind and
00037   defaults.  PgOptions will process the command line and
00038   then the program can query what was set.
00039 
00040   Like standard unix programs, an option of "--" ends option
00041   parsing.  All the remaning arguments are args to the
00042   program.
00043 
00044   There are three kinds of options: BOOL, DOUBLE and STRING.
00045   All three kinds may have a value appended to the option name with "=".
00046   (Like: "--double-example=10.5", "--output-dir=foo".)
00047 
00048   BOOL values may have a prefix of "-no-" or "--no-" to set
00049   the value to "false".  If the arg after a bool option is
00050   "true/1" or "false/0" it is used as the value.  Otherwise
00051   the option is set to true.  (Note that this might make
00052   working with a file called "true" ambiguous.  In that case
00053   use "--" to seperate the args from the option.)
00054 
00055   Non-option args to the program are put into the "args"
00056   list which the program can retreive with "getArg(idx)".
00057 
00058   @section PgOptionsExample Example
00059 
00060   Normal usage looks like:
00061 
00062 @verbatim
00063 int main (int argc,char* argv[]) {
00064 
00065   PgOptions opts;
00066   opts->setUsage("A nice paragraph about what this program does.");
00067   opts->defineOption("p","print",PgOpt::BOOL_OPT,
00068                      "Do you want to print the output?",
00069                      "true");
00070 
00071   opts->parseArgv(argv);
00072 
00073   if (opts->getBool("print")) {
00074     doPrint();
00075   }
00076 }
00077 @endverbatim
00078 
00079   @section PgOptionsNotes Notes
00080 
00081   * Please try and write the options in the "positive sense".
00082   If the default is to print, then define the option "print"
00083   to be true.  The user can use "--no-print" when they dont
00084   want to.  (Dont define "--no-print".)
00085 
00086  */
